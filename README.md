# EvidenciasAlgoritmia
Repositorio con 25 problemas para el curso universitario de Algoritmia, elaborados en PSeInt.

Se eligio este compendio de problemas para reflejar los conocimientos basicos que deberian adquririrse dentro del curso de Algoritmia.
Se adicionaron los algoritmos bidimensionales con un enfoque en estructuras matematicas complejas y sus propiedades.
